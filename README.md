# 오리온 홈페이지 연습
React, Typescript 퍼블리싱 연습 <br/>
[오리온 홈페이지](https://www.orionworld.com/)를 참고하여 퍼블리싱 연습 <br/>
<br/>

## 시작
```
npm i
npm start
```
<br/>

## 사용 스택
`React` `Typescript` `styled-coponents` `msw`

<br/>

## 기간
* 22.12.19(월), 22.09.21(수) ~ 22.12.26(목) (주말 제외 총 6일)
